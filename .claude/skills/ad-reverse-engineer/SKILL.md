# Ad Reverse Engineering Framework

## Public Ad Sources

### Meta Ad Library (Facebook/Instagram)
- URL: `https://www.facebook.com/ads/library/`
- Search by: Page name, keyword, or advertiser
- Available data: Ad creative, copy, launch date, active status, platforms

### Google Ads Transparency Center
- URL: `https://adstransparency.google.com/`
- Search by: Advertiser name
- Available data: Search ads, display ads, YouTube ads, date ranges

### LinkedIn Ad Library
- URL: `https://www.linkedin.com/ad-library/`
- Available data: Sponsored content, ad copy, targeting regions

## Analysis Framework

### 1. Ad Creative Analysis
- **Visual style**: Colors, imagery, faces vs. product, UGC vs. polished
- **Format**: Static, carousel, video, stories
- **Thumbnail/hook frame**: What stops the scroll?

### 2. Copy Analysis
- **Hook type**: Question, bold claim, pain point, curiosity gap
- **Body structure**: Problem-agitate-solve, story, social proof
- **CTA language**: What action do they want?
- **Tone**: Formal, casual, urgent, educational

### 3. Offer Analysis
- **Primary offer**: What are they selling?
- **Incentive**: Discount, bonus, guarantee, scarcity
- **Friction reducers**: Free trial, money-back, no commitment

### 4. Volume & Variation Patterns
- **Active ad count**: How many ads running simultaneously?
- **Variation testing**: Same offer with different hooks?
- **Longevity**: Which ads have run longest (likely winners)?

## Output Structure

Deliver findings in this format:

```
## [Company Name] Ad Strategy Analysis

### Executive Summary
- Primary advertising platforms
- Estimated ad volume/intensity
- Core messaging theme

### Ad Creative Breakdown
[3-5 representative ads with analysis]

### Copy Patterns
- Winning hooks identified
- CTA patterns
- Tone/voice characteristics

### Offer Structure
- Main offer(s)
- Pricing psychology tactics
- Trust/risk reversal elements

### Strategic Insights
- What's working (run longest/most variations)
- Gaps or opportunities
- Replicable tactics for your use
```
