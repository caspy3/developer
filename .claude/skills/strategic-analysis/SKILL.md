# Strategic Analysis Skill

## Purpose
Apply rigorous analytical frameworks to inform strategic decisions and evaluate options.

## Analysis Frameworks

### Decision Matrix

```

### Decision Context
What triggered this decision? Why now?

### Decision Criteria
| Criterion | Weight (1-5) | Description |
|-----------|--------------|-------------|
| [Criterion 1] | [Weight] | [What it means] |
| [Criterion 2] | [Weight] | [What it means] |

### Options Evaluation
| Criterion | Weight | Option A | Option B | Option C |
|-----------|--------|----------|----------|----------|
| [Criterion] | [Wt] | [Score 1-5] | [Score] | [Score] |
| [Criterion] | [Wt] | [Score] | [Score] | [Score] |
| **Weighted Total** | | [Sum] | [Sum] | [Sum] |

### Qualitative Considerations
```

## Analysis Quality Standards
1. **Assumptions explicit**: State what must be true
2. **Confidence noted**: Distinguish certainty from estimates
3. **Alternatives considered**: Not just the preferred option
4. **Sensitivity tested**: How conclusions change with inputs
5. **Actionable output**: Clear recommendation with rationale
