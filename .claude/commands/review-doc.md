# Document Review Workflow

Review the provided document and analyze:

1. **Summary**: What is this document about? (2-3 sentences)
2. **Key Points**: What are the 5 most important takeaways?
3. **Questions**: What clarifying questions would you ask the author?
4. **Action Items**: What actions does this document require?

Format your response with clear headers for each section.
